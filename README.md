# led_fan

[![Build Status](https://travis-ci.org/hmgle/led_fan.png?branch=master)](https://travis-ci.org/hmgle/led_fan)

无需花钱买硬件, led_fan 让你在软件上拥有一个闪烁的 [LED 风扇(摇摇棒)](https://www.google.com/search?q=led+风扇&tbm=isch)

![demo.gif](demo.gif)

依赖的库:
- SDL2
- SDL2_gfx

## 编译及运行

```
make
./led_fan "需要显示的字符"
# 按 'q' 终止
./led_fan # 不带参数时跑一个彩虹猫~
```
